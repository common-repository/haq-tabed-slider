== Description ==
Allows you to add fully customizable, responsive. This plugin creates an image slide in your theme. Build beautiful image slider, You can upload/delete images via the admin panel,
Every slide can also add Title and Description. Use shortcode  [tabed_slider id="123"] for page, post, widget etc.

== Installation ==

1. Upload `haq-tabed-slider.zip` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Shortcode show every post according to post ID.
1. Use the provided shortcodes [tabed_slider id="123"] anywhere on your website.

Just create a normal WordPress page, and use the following shortcodes:
	
* For slider: `[tabed_slider id="123"]`

== Frequently Asked Questions ==

<strong>Question:</strong> How do I add the slider to my site? <br>
<strong>Answer:</strong> You can add your slider by shortcode on page, post or widget text or PHP shortcode.

<strong>Question:</strong> Any affect on other Files?<br>
